List tags of HTML

Order List

Create the Order list

Syntax:
 <ol> <li> "item" </li> </ol>
Output:
  1. Pune
  2. Mumbai
  3. Nashik

Changing the number of order list using start

Syntax:
 <ol start="4" > </ol>
Output:
  1. Pune
  2. Mumbai
  3. Nashik

Reverse the numbering by adding the reversed in ol element

Syntax:
 <ol reversed > </ol>
Output:
  1. Pune
  2. Mumbai
  3. Nashik

Changing the type of numeral

Type Description Example
1 Default value-Decimal numbes 1,2,3
a Alphabetically ordered(lowercase) a,b,c,d,e
A Alphabetically ordered(uppercase) A,B,C
i Roman Numberal(lowercase) i,ii,iii,iv
I Roman Numberal(Uppercase) I,II,III,IV
Syntax:
 <ol type="1|a|A|i|I" > </ol>
Output for type ='a':
  1. Pune
  2. Mumbai
  3. Nashik
Output for type ='A':
  1. Pune
  2. Mumbai
  3. Nashik
Output for type ='i':
  1. Pune
  2. Mumbai
  3. Nashik
Output for type ='I':
  1. Pune
  2. Mumbai
  3. Nashik

Create the Unordered list

NestedList List

Create the NestedList list

Syntax:
 <ul>
	<li> "item1" </li> 
          <ul> <li> "item2" </li> </ul> 
 </ul>
Output:
  • Pune
    • Kothrud
    • Deccan Corner
    • Shivaji Nagar
  • Mumbai
  • Nashik

Description List

Create the Description list

Syntax:
 <dl> 
	<dt> "item" </dt>		
        <dt> "item" </dt>	
</dl>
Output:
Football
Football required 11 players for playing
Chess
